# FLASK

## 모르는 단어
>+ ### __URI__
>    + url보다 uri가 상위 개념이다. 
>    + exam) URI가 동물이면 URL은 강아지가 된다.
>+ ### __instance__
>    + instance는 클래스에 속하는 각 객체 이다
>       + ex) list가 있는데 mylist라는 객체를 생성한다.
>+ ### __logging__
>    + 로그를 기록하는 것이다.
